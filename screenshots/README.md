# Screenshots

Drop the following PNGs in this folder and the gallery in the main
[`README.md`](../README.md#screenshots) will render (uncomment the table there).

| File | What to capture |
|------|-----------------|
| `home.png` | The home page hero ("You've felt it. Another language named it.") with the feeling-search box |
| `name-my-feeling.png` | A Name My Feeling result — e.g. input "wanting to visit a place you've never been" → **Kaukokaipuu**, showing the explanation + cross-language alternatives |
| `word-portrait.png` | A word portrait (e.g. **Saudade**) showing the six-dimension grid |
| `language-map.png` | The `#/map` force-directed language network |

**How to capture (Windows):** open <https://project-babel-five.vercel.app>, press `Win + Shift + S`, select the area, then save with the filename above. Recommended width ~1200px.

**How to add them to the repo:** either drop the files here and `git push`, or drag-and-drop them into this folder via the GitHub web UI (which commits them automatically).
